# umi project
react+antd+dva+umi 一个中台的纯模板
## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```
## router
路由分为登录页以及应用页两块，路径在：config/routes.ts
